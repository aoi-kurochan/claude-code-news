# Claude Code ニュースレポート 2026-04-10

## 最新ニュース・アップデート

### バージョン 2.1.89〜2.1.92（2026年4月）の主な変更点

- **`/powerup` インタラクティブレッスン**機能が追加。コマンドラインからチュートリアルを受けられるように。
- **MCP 500K 結果ストレージ**：MCP ツールの大容量レスポンスをキャッシュする仕組みを導入。
- **権限の遅延決定（Defer Permission Decisions）**：セキュリティポリシーを柔軟に運用できる新設定。
- **`/cost` にモデル別・キャッシュヒット別の内訳表示**が追加（サブスクリプションユーザー向け）。
- **チラつきのないalt-screenレンダリング**：`CLAUDE_CODE_NO_FLICKER=1` 環境変数で有効化可能。
- **`PermissionDenied` フック**：自動モードのクラシファイア拒否時にも起動するフックが追加。
- **`@メンション` での名前付きサブエージェント補完**が可能に。
- **Bedrock セットアップウィザード**：ログイン画面から AWS 認証・リージョン・モデル設定を対話的に設定可能。
- **大容量ファイル書き込みが60%高速化**（Write ツールの差分計算改善）。
- **`/release-notes` がバージョン選択インターフェース**に進化。

### Claude Code Analytics API の提供開始

Anthropic が **Claude Code Analytics API** を公開。組織単位で日別・集計した Claude Code の利用メトリクスをプログラムからアクセスできるようになった。大規模な開発チームの利用状況把握や予算管理に活用できる。

### OpenClaw 連携の有料化（2026年4月4日〜）

2026年4月4日 正午（太平洋時間）以降、Claude サブスクリプションの利用枠を OpenClaw などのサードパーティハーネスで消費できなくなった。別途費用が必要。

### サービス障害（2026年4月6日）

4月6日、Claude.ai および Claude Code でログインエラーが多発。ボイスモードなど一部機能にも影響が出た。

### プロンプトインジェクション脆弱性の修正

ソースコードの流出により、プロンプトインジェクション脆弱性が発覚。Anthropic は迅速にパッチを適用した。

## 活用事例・Tips

### 業務自動化の実績

- **営業部門の月次集計**：手作業で平均3.2時間かかっていた Excel 集計を Claude Code に委任→**14分に短縮（93%削減）**。
- **バグ修正の効率化**：原因特定を約15分、修正〜テストまでの一連作業を従来の半日から**約1.5時間**に短縮。
- **契約書管理**：複数取引先の契約書を整理し、契約管理台帳を自動作成。AIレビューで見落としを**95%削減**した事例も。

### おすすめ Tips（2026年版）

1. **`/clear` を積極的に使う**：コンテキストウィンドウが埋まると性能が低下するため、新しいタスクを始めるたびにクリアする。
2. **Plan Mode を活用する**：実装前に Claude に探索・計画だけさせてから、auto-accept 編集モードで実装を開始すると思考のズレが減る。
3. **`claude update latest` を定期実行**：マイナーリリースでも大きな改善が含まれることが多い。
4. **Skills（スキルファイル）を活用**：CLAUDE.md とは異なり、必要なときだけロードされるため、コンテキストをリーンに保てる。
5. **`Esc+Esc` でチェックポイント一覧を表示**：過去の状態に戻りたいときに便利。
6. **`/context` でコンテキスト使用状況を確認**：肥大化してきたらタイムリーに `/compact` を使う。
