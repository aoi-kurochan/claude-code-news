# Claude Code ニュースレポート 2026-04-06

## 最新ニュース・アップデート

### April 2026 の主な変更点

- **フリッカーフリーのalt-screenレンダリング**を追加し、UI表示が改善
- **named subagents の @ メンション**対応
- **`/powerup`コマンド**を追加 — Claude Codeの機能をアニメーションデモ付きで学べるインタラクティブなレッスン
- **Bedrockセットアップウィザード**を追加 — AWSの認証、リージョン設定、認証情報確認、モデルのピン留めをガイド
- **`/cost`コマンドの強化** — サブスクリプションユーザー向けにモデル別・キャッシュヒット別の内訳を表示
- **Write ツールのパフォーマンス改善** — タブ/`&`/`$`を含む大きなファイルでdiff計算が60%高速化

### March 2026 の主な変更点

- **Computer Use** — Mac アプリをAPIなしで操作可能に（スクリーンショットを撮影し、画面を把握してマウス・キーボード操作を実行）
- **100万トークンのコンテキストウィンドウ**対応で大規模プロジェクトにも対応
- **`--bare`フラグ**を追加（スクリプトからの呼び出し向け）
- **`--channels`パーミッションリレー** — 承認プロンプトをスマートフォンに転送可能
- **Auto Mode** — Web・モバイルセッションでPRのフォロー、CIの失敗修正、コメント対応をクラウドで自動化
- インタラクティブなチャート、図、ビジュアライゼーション、モバイルアプリの生成が可能に

### 3月31日 ソースコード流出インシデント

2026年3月31日、Anthropicがnpmレジストリにバージョン2.1.88をリリースした際、59.8MBのJavaScriptソースマップファイルが誤って含まれ、約512,000行の難読化されていないTypeScriptコードが公開されるという事態が発生。AI業界で最も議論されたサプライチェーンインシデントの一つとなった。

### ポリシー変更（4月4日）

Anthropicがサブスクリプション利用者に通知: サブスクリプション枠をOpenClawなどのサードパーティツールに適用できなくなり、従量課金制（pay-as-you-go）を利用する必要あり。

### 実践Tips

- **CLAUDE.md はコンパクトに** — Claude は毎回読むため、ビルドコマンド・アーキテクチャ概要・コーディング規約など「コードから読み取れない情報」に絞って記述すること。
- **`Shift+Tab` でプランモード** — 変更を依頼する前にプランモードに切り替え、Claude が読み取り・分析のみ行う状態でプランを確認。承認後に実装へ移行する。
- **`Esc` で中断、`Esc+Esc`（または `/rewind`）でロールバック** — コンテキストを失わずに任意のチェックポイントに戻れる。
- **並列エージェントチームの活用** — 複数の並列エージェントが共有タスクリストを通じて連携。コードレビューの多角的分析や、大規模リファクタリングのファイルセット分割処理に活用できる。
- **`/config` でスタイル設定** — Explanatory（詳細なステップ解説）、Concise（簡潔なアクション重視）、Technical（専門的・正確）などのスタイルから選択可能。
