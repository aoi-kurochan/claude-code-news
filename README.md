# Claude Code News

Claude Codeの最新活用事例やニュースを毎日自動収集し、日本語でまとめたレポートを配信するリポジトリです。

## 概要

- 毎朝7時にClaude Codeが自動でウェブ検索を実行
- 最新のClaude Code関連ニュース・活用事例を収集
- 日本語のMarkdownレポートとして `reports/` ディレクトリに保存
- 自動でこのリポジトリにコミット
